CSS Snippet Cheat Sheet

A place to store all of my favorite CSS code snippets and a related article
Flexbox Row

Use these three properties to create a Flexbox row layout.

.row {
  display:flex;
  flex-direction: row;
  flex-wrap: wrap;
}

Flexbox Column

Use this to create a Flexbox column layout.

.column {
  display:flex;
  flex-direction: column;
}

CSS Grid Layout

Build a 12-column layout using css grid

.grid{
  display: grid;
  width: 100%;
  grid-template-columns: repeat(12, 1fr);
}

Linear Gradiants

This will create a background linear gradient from top to bottom

.linear-gradient-background{
  background-image: linear-gradient(
    rgb(214, 3, 168) 0%,
    rgba(214, 3, 168, 0.534) 100%
  );
}

Box Transition Glow

Use transition and box shadows to glow on hover

.code-card .card-header{
  border-radius: 8px;
  transition: all 0.5s ease-in-out;
}

.code-card:hover,
.code-card:hover .card-header{
  box-shadow:inset 0px 0px 8px
    rgb(214, 3, 168) 0 0 15px,
    rgba(214, 3, 168, 0.534)
}

Overlay Card with Title

Use postion properties and a negative margins to raise elements higher than their natural starting point.

.card-header{
  position: relative;
  margin-top: -20px;
}
